Servlets examples

A collection of code Servlet examples. Please look at the source code (especially the use of cookies, sessions, forward, include and redirect.

  1. bookreview: examples from the course material, made of a HTML file and a servlet to process a form. The ACTION of the form in the HTML file is set to the Servlet.
  2. bookreview 2 : same example built only with one servlet that handles GET request to display the form and POST request to process the form.
  3. cookies test: set cookies and displaying all cookies
  4. using the ServletContext to store data available from another Servlet
  5. Sessions: a login example using the RequestDispatcher include file function.
  6. Dispatcher: an example of forwarding a request to another Servlet (/Servlets/DispatcherServlet?userName=servlet)
  7. a shopping cart servlet based on cookies
  8. a servlet returning JSON data
Author: Solange Karsenty.
